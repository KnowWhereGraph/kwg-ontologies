Overview of the raw dataset back to ToC

  • Name of dataset:US district court jurisdictions
  • Source Agency: Federal Judicial Center
  • Data source location: US district court jurisdictions (explore)
  • Metadata description: US district court jurisdictions (about)
  • Type of data source: Shape file (CSV, KML, GeoJSON, File Geodatabase also supported)
