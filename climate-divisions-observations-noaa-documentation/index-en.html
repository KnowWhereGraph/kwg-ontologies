Overview of the raw dataset back to ToC

  • Name of dataset: Climate division boundaries and climate observations
  • Source Agency: NOAA
  • Data source location: climate division boundary shapefile
  • Metadata description: climate division boundary metadata
  • Data source location: climate observations directory
  • Metadata description: climate division observations metadata
  • Temporal update frequency (observation data): monthly
  • Reference system: NAD83 (requires transformation to WGS84 using geopandas)

Climate Divisionc back to ToC or Class ToC

IRI: http://stko-kwg.geog.ucsb.edu/lod/ontology/ClimateDivision

This class denotes climate divisions in the NOAA CONUS climate divisional dataset. Climate division used to generate climate analyses for the contiguous United States (CONUS).
