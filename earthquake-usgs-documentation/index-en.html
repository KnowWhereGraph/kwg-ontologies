Overview of the raw dataset back to ToC

  • Name of dataset: ANSS Comprehensive Earthquake Catalog (ComCat)
  • Source Agency: USGS
  • Data source location: API Documentation - Earthquake Catalog
  • Metadata documentation: ComCat Documentation
  • Type of data source: API
  • Format of data returned:CSV (GeoJSON, KML, xml, text also supported)
  • Data update interval:real-time
