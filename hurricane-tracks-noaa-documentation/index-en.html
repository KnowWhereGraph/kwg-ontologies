Overview of the raw dataset back to ToC

  • Name of dataset:Historical hurricane tracks
  • Source Agency: NOAA
  • Data source location: Query stormIDs by polygon,Fetch JSON file by stormID
  • Metadata description: NHC data archive documentation
  • Type of data source: API
  • Format of data returned:JSON
  • Data update interval: yearly
