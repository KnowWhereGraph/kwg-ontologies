Ontology Specification Draft

Overview of the raw dataset back to ToC

  • Name of dataset: USDA NASS Cropland
  • Source Agency: USDA NASS
  • Data source location: CropScape Data Layers
  • Metadata description: Developer Guide
  • Type of data source: API
  • Format of data returned: .tif files (or tabular)
  • Data update interval: annually
  • Volume of raw data: extremely Large
